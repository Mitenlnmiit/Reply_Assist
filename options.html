Reply Assist

Configure your AI-powered reply refinement settings

API Configuration

Get your free API key from Google AI Studio. Need help? Watch this quick guide: How to get a Gemini API key.

How to Use

  1. Set your Gemini API key above.
  2. Navigate to any website with a text input field.
  3. Click in a text input field and type your message.
  4. Use ALT+Q to view a refined version of your text.
  5. Use ALT+X to provide custom instructions for refining the text.

Features

  • Works on all major websites, such as Reddit, LinkedIn, YouTube, Slack, WhatsApp, Gmail,Twitter and Google Forms
  • Context-aware message refinement
  • No data stored locally or on servers
  • Privacy-focused (uses your own API key)
  • Keyboard shortcuts for quick access

Privacy & Security

This extension prioritizes your privacy:

  • No data storage: Your messages are only sent to Google's Gemini API
  • Your API key: You control your own API key and usage
  • No tracking: No analytics or user tracking
  • Local processing: All text extraction happens in your browser

Troubleshooting

Common Issues:

  • Extension not working: Make sure you're on a supported website and have clicked in a text field.
  • API errors: Check that your API key is correct and has sufficient quota.
  • Keyboard shortcuts not working: Try refreshing the page or check for conflicts with other extensions.
  • No conversation context: The extension works best on pages with clear message structures.
  • Enable Clipboard Permissions: The extension uses the clipboard on some websites to enable direct pasting. Enabling clipboard permissions will enhance your user experience.
