How to Use

  1. Set your Gemini API key above.
  2. Navigate to any website with a text input field.
  3. Click in a text input field and type your message.
  4. Use ALT+Q to view a refined version of your text.
  5. Use ALT+X to provide custom instructions for refining the text.

Features

  • Works on common sites including Reddit, LinkedIn, Gmail, and Google Forms.
  • Context-aware message refinement.
  • No analytics or telemetry collected.
  • Uses your API key; messages go only to the model API.
  • Keyboard shortcuts for quick access.

Privacy & Security

Privacy details:

  • No data storage: The extension does not store message content.
  • Your API key: You control your own API key and usage
  • No tracking: No analytics or user tracking.
  • Local processing: Text extraction happens in your browser.

Troubleshooting

Common issues

  • Extension not working: Make sure you're on a supported website and have clicked in a text field.
  • API errors: Check that your API key is correct and has sufficient quota.
  • Keyboard shortcuts not working: Try refreshing the page or check for conflicts with other extensions.
  • No conversation context: The extension works best on pages with clear message structures.
  • Clipboard permissions: Some sites require clipboard access to enable direct pasting.
